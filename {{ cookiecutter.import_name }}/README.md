## Development

### Prereqs

Poetry
https://python-poetry.org/docs/

Brew packages
```
brew install pyenv md5sha1sum libxml2 libxmlsec1 pkg-config postgresql \
    openssl readline sqlite3 xz zlib tcl-tk
```

Then, from the project root
```
pyenv install
```

### Project Install

```
make install
```
